# Java Swing Case Studies

This repository contains Java **Swing-based case study applications** developed as part of academic coursework.  
The projects focus on building simple GUI-based desktop applications while applying core Java and object-oriented programming concepts.

## Key Highlights
- Developed multiple Java Swing GUI applications
- Designed user interfaces using Swing components
- Implemented event handling for user interactions
- Followed modular and structured coding practices

## Concepts & Technologies Used
- Java (Core Java)
- Java Swing (GUI)
- Event Handling
- Object-Oriented Programming (OOP)
- Control Statements

## Tools & Environment
- Java
- VS Code / IntelliJ IDEA
- Git & GitHub

## How to Run
1. Clone the repository  
2. Open the project in any Java IDE  
3. Run the main `.java` file containing the Swing frame

## Purpose
Developed for academic learning to gain hands-on experience in Java Swing and GUI-based application development.
